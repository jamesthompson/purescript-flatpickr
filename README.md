# purescript-flatpickr

PureScript bindings for [flatpickr](https://chmln.github.io/flatpickr/).

## Run the example

``` shell
$ bower i
$ pulp browserify --include example/src --to example/dist/bundle.js
$ google-chrome example/dist/index.html
```
